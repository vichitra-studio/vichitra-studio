# Vichitra Collective

Vichitra is a Sanskrit word which could mean bizarre, strange, or unique, among many other
things. Vichitra Collective is an attempt to flow with uncertainty instead of fighting it,
practiced across photography, software, and a few other things in between. [Read the full story](https://blog.vichitracollective.com/but-what-is-vichitra-collective/).

**Building**
- [`devspec_toolkit`](https://github.com/vichitra-studio/devspec_toolkit): a schema-first, AI-assisted workflow that turns spec into implementation as a deterministic, machine-checkable pipeline. Actively releasing (v1.1.1); it's the toolkit running my own projects, including this one.
- [`vc_image_utils`](https://github.com/vichitra-studio/vc_image_utils): early-stage RAW/HDR image-processing tooling, growing out of a decades-long photography practice.
